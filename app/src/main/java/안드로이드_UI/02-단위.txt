XML로 화면을 그릴때 객체의 크기를 지정하는 단위

단위
픽셀(Px,Pixel)
- 실제 존재하는 물리적인 단위
- 같은 단위면적에 픽셀수가 많을 수록 해상도가 높다
- 픽셀은 물리적인 화면을 구성하는 최소의 단위
- 화면의크기(해상도)가 다르면 객체의 크기가 각각이다.

Dp(Density-Independant Pixels)
- 픽셀과 무관한 단위
- 화면의 크기가 달라도 동일한 비율로 객체를 보여주기 위한 안드로이드에서 정의한 단위
- 주로 이미지/텍스트등의 크기지정

Sp(Scale-Independant Pixels)
- 시스템설정(OS)에 영향을 받는 단위, 설정에서 글꼴 크기 조정시 영향을 받음
- 주로 텍스트 크기를 지정하기 위해 사용되는 단위
- 실지로를 잘 안쓰인다. 구글은 권장

DPI(dots per inch)
- 100dp : 1인치당 100픽셀 존재(1dp==1px)
- ldpi : 120dpi
- mdpi : 160dpi(기본)
- hdpi : 240dpi
- xhdpi : 320dpi
- xxhdpi : 480dpi
- xxxhdpi : 640dpi
- 이미지의 크기를 표시하는 또 하나의 단위
