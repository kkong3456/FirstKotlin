Room
- 데이터베이스를 쉽게 사용하기 위한 API

안드로이드 데이터 저장방식
- key-value : sharedPreference (내장)
- 관계형 방식 : SQLite (내장) ==> SQLite API를 통해 직접 사용하는것 보다 Room을 이용하여 SQLite에 접근한다. 권장사

안드로이드의 내장된 관계형 데이터베이스를 사용할때 장점
- 요청 : 앱 -> 서버 -> 데이터베이스(관계형)
- 응답 : 데이터베이스 -> 서버 -> 앱(관계형 데이터베이스)
- 원격에 있는(인스타그램) 데이터(사진)를 내 폰에 저장 할 수있고, 오프라인인 경우에도  업데이트는 안되지만 저장되어 있는 기능(사진보기등)을 사용할수 있다.(캐시)
- 캐싱을 통해서 데이터를 캐시해 놓아야만 중복으로 다운로드 받지 않게 되어 데이터 요금을 아낄수 있다.
