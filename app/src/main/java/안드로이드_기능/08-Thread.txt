Main Thread
- 최초로 실행되며 무조건 실행된다.
- 두번째 쓰레드는 무조건 메인 쓰레드에 의해서 실행이 되었다.
- 세번째 쓰레드는메인 혹은 두번째 쓰레드에 의해서 실행이 된다.
