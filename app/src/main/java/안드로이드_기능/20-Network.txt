Serializing(Serializer)
- JSON으로 데이터를 교환할때
- 꼬챙이로 객체를 꿰어서 서버에 던진다.
- 학생객체(이름,나이,번호..), 이름+나이+번호를 꼬챙이로(직렬화) 꿰서 던진다.
- 객체 전송을 위한 데이터 포맷을 변경
- 클라이언트의 학생객체(JSON)  ---> 클라이언트(시리얼라이징) --> 전송 --> 서버 -->서버(역시리얼라이징)->서버(학생객체 JSON)
